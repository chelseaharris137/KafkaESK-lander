Kafka Streams Visualization Tool

See your Kafka topic message streams in real time

 
 
 
 

Features

Dashboard Display

Vizualize your data via a multitude of live graph displays within the dashboard.

Airtable

Assign individual team members to project tasks using Airtable.

Websockets

Experience fast, continuous streaming of data with server and client-side Websocket connections.

 




KafkaJS & KSQL

Our server is built using KafkaJS and our stream queries utilize KSQL. So Javascript developers familiar with SQL syntax can easily implement our tool without having to learn additional languages.

Mock Data

KafkaESK is pre-built with mock data so you can run the app right out of the box.
